# brainfuck
A brainfuck interpreter

## Usage

```bash
# Execute passed source
brainfuck -e "+[-[<<[+[--->]-[<<<]]]>>>-]>-.---.>..>.<<<<-.<+.>>>>>.>.<<.<-."
```

```bash
# Execute source file
brainfuck ./file.bf
```

```bash
# Execute source from stdin
brainfuck
```
